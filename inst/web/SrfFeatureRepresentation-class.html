Description

Surface ('SRF') feature representation

Slots

NAME_STR
private 'SRF'

N_I
The maximum value $N$ for which $n$-grams will be extracted from object string sequences, with $n \in \{1,...,N\}$.

OBJECT_SEPARATOR_STR
The separator used when collapsing strings representing single observations into object string sequences (e.g., texts).

SRILM_NGRAMCOUNT_PARAMETERS_CVEC
Extra command line parameters to SRILM ngram-count. This utility is run to extract features.

SRILM_NGRAM_PARAMETERS_CVEC
Extra command line parameters to SRILM ngram. This command is run to calculate feature scores.

XPATH_STR
XPath used to extract the surface strings in the XML documents that represent observations. It ought to point to a text node.
