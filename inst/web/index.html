Extracts from a collection of XML-encoded documents a (sparse) design matrix with variables for each observed surface (word n-grams) and structural feature of an objects. Objects can be represented by one or more observations (rows) in an annotations table. Such objects can f.i. represent text segments or single sentences, linguistic sentence graph (e.g., a syntax tree). The structural feature representation of objects, linguistic networks, are unions of the XML documents. These unions are multidigraphs with arcs labeled with attribute (name, value) pairs of elements. For example, arcs can be considered labeled with linguistic dependency relations between words in the syntax tree. Trails in these linguistic networks are randomly sampled, and then treated as sequences of arcs from parent to child elements, from which n-grams can be extracted as well.
