Usage

FEATURES_EXTRACTION_PARAMETERS
See: FeaturesExtractionParameters.
FEATURE_REPRESENTATIONS_LST
See: SrfFeatureRepresentation, StrctFeatureRepresentation.

The DESIGN_DMAT or design matrix with feature scores, with objects as rows indexed by their object_ID and named columns as features.
