Parameters of the feature extraction job

Description

Parameters of the feature extraction job

Slots

ANNOTATIONS_TABLE_FILE_PATH
The file path to the annotations table. See: Annotations, metadata_write_annotations_table_template, metadata_verify_annotation_table_consistency, metadata_read_annotations_table.

DOCUMENTS_DIR_PATH
The root directory path containing the XML documents representing observations. To this the relative_file_path column in annotation tables must be relative.

JOB_NAME_STR
A name for the particular feature extraction job.

OUTPUT_DIR_PATH
The root directory path for any filesystem output.
